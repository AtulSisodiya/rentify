# Rentify - Where Renting Meets Simplicity

## Introduction
Rentify is a web application designed to simplify the rental process for both property owners and tenants. Owners can list their properties while tenants can browse through available listings and express their interest.

## Features
### Part I: Basic Application
- **User Registration and Login**
  - Register as either a seller or a buyer.
  - Login to access personalized features.
- **Seller Flow**
  - Post properties with detailed attributes.
  - View, update, and delete posted properties.
- **Buyer Flow**
  - View all listed properties.
  - Express interest in properties to view seller details.
  - Apply filters to narrow down property search.

### Part II: Advanced Features
- **Pagination**: Handle large datasets efficiently.
- **Form Validation**: Ensure proper data submission.
- **Mandatory Login for Viewing Seller Details**: Secure access to seller information.
- **Like Button**: Track and display live count of likes for each property.
- **Email Notifications**: Notify both buyers and sellers when interest is expressed.

## Technology Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Java, Spring Boot

- Java 11+
- Maven
- Node.js (if additional JavaScript tools are used)
